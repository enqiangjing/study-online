# 线上视频课程学习系统



## 1. 系统介绍

* 在线学习管理系统，核心功能为课程管理及课程视频学习，其他包括
* 建立多个前端项目，实现多终端使用（PC Web、Mobile、小程序）
* **详细使用请见** [**操作手册**](https://github.com/enqiangjing/study-online-web/blob/main/options.md)
